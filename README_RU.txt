TELEGRAM-БОТ 9ROUTER V4
=========================

НОВОЕ В V4
-----------
1. Inline Mode:
   В любом чате Telegram можно написать:

   @username_бота объясни простыми словами, что такое API

   После этого нужно выбрать появившийся результат «Ответ ИИ».

2. Упоминания в группах:
   Добавь бота в группу и напиши, например:

   @username_бота объясни эту ошибку

3. Объяснение сообщения другого человека:
   Ответь на нужное сообщение и напиши обычное сообщение с упоминанием бота:

   Объясни, что человек хотел сказать, @username_бота

   Бот получит текст сообщения, на которое ты ответил, и учтёт его в ответе.

ВАЖНОЕ ОТЛИЧИЕ INLINE И ОТВЕТА В ГРУППЕ
----------------------------------------
Inline-запрос @бот вопрос получает только написанный вопрос. Он не видит
сообщение, на которое пользователь ответил.

Чтобы бот увидел исходное сообщение, он должен быть добавлен в группу.
Ответь на сообщение и упомяни бота в обычном сообщении. Лучше ставить
упоминание не первым, чтобы Telegram не открыл Inline Mode автоматически:

Объясни это, @username_бота

КАК ВКЛЮЧИТЬ INLINE MODE
------------------------
1. Открой @BotFather.
2. Отправь /setinline.
3. Выбери своего бота.
4. Укажи подсказку, например:

Напиши вопрос для ИИ…

Без этого шага Telegram не будет отправлять боту inline_query.

GUEST MODE — ВЫЗОВ БЕЗ ДОБАВЛЕНИЯ БОТА В ЧАТ
------------------------------------------------
В новых версиях Telegram бот может получить упоминание и ответить в чате,
даже если он не добавлен участником. Для этого:

1. Открой @BotFather.
2. Открой настройки нужного бота.
3. Включи Guest Mode.

После этого можно ответить на сообщение человека и написать, например:

Объясни, что он хотел сказать, @username_бота

В Guest Mode бот может отправить текстовый ответ. Создание и отправка локальных
файлов остаётся доступным в личном чате с ботом или в группе, куда бот добавлен.

ЗАПУСК
------
1. Вставь токен Telegram-бота в .env.
2. Запусти 9Router.
3. Установи зависимости:

npm install

4. Проверь код:

node --check bot.js

5. Запусти:

npm start

PM2:

pm2 start bot.js --name 9router-bot
pm2 save

После обновления старого бота:

npm install
pm2 restart 9router-bot

ОСТАЛЬНЫЕ ФУНКЦИИ
-----------------
- Кнопки управления вместо списка команд.
- Три модели: ChatGPT 5.5, Codex и третья доступная GPT-модель.
- Фиксированные безопасные системные правила.
- Форматирование Telegram: цитаты, жирный, курсив и код.
- Один созданный файл отправляется документом.
- Несколько файлов сайта или проекта собираются в ZIP.
- Ограничение доступа через ALLOWED_USER_IDS.

БЕЗОПАСНОСТЬ
------------
Ключ 9Router ранее отправлялся в переписке. После тестирования рекомендуется
создать новый ключ и заменить NINEROUTER_API_KEY в .env.
